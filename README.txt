ЭТО СТАРЫЙ РЕПОЗИТОРИЙ С БИЛДАМИ
Этот репозиторий я создал, когда сам git был для меня новинкой. Тогда я решил для себя, что
один репозиторий для всех игр - отличное решение. Потом появились .apk- и .exe-версии, для
которых были созданы отедльные ветви для удобства. Тут я понял, что это очень грязное дело
- класть всё и вся в один репоз. В общем, теперь у каждой игры будет своей репозиторий,
чтобы всё выглядело как надо. До обновы :)
